# Nagakirthi Professional Portfolio Website

A modern, responsive portfolio website template based on professional CV/Resume design patterns.

## Files Included

1. **index.html** - Main HTML structure
2. **styles.css** - Professional styling with responsive design
3. **script.js** - Interactive features and smooth scrolling
4. **README.md** - This file

## Design Features

- **Modern Professional Design**: Clean, minimalist layout with professional color scheme (dark blue #2c3e50, accent red #e74c3c)
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations**: Fade-in effects and hover animations
- **Easy Navigation**: Sticky navigation bar with smooth scrolling
- **Well-Organized Sections**:
  - Hero/Header section
  - About Me
  - Professional Experience
  - Skills & Expertise
  - Featured Projects
  - Education
  - Contact Information

## How to Customize with Your Resume

### 1. Update Personal Information
Open `index.html` and replace:
- "Nagakirthi" with your full name
- "Professional Portfolio" with your professional title
- Update all placeholder text in each section

### 2. About Section
- Update the "About Me" paragraph with your professional summary (2-3 paragraphs)
- Add your unique value proposition

### 3. Experience Section
- Replace placeholder job titles with your actual positions
- Update company names and locations
- Add employment dates
- List key achievements and responsibilities

### 4. Skills Section
Replace placeholder skills with your actual skills:
- Technical Skills (Programming languages, tools, technologies)
- Professional Skills (Soft skills, methodologies, certifications)

### 5. Projects Section
Add your featured projects:
- Update project titles
- Write brief descriptions
- Add relevant technologies/tools used
- Optional: Replace the gradient background image with actual project screenshots

### 6. Education Section
- Add your degree names
- Update institution names
- Add graduation years

### 7. Contact Information
Update contact links:
- Replace email address
- Add your LinkedIn URL
- Add your GitHub profile URL
- Add any other relevant links (Portfolio, Blog, etc.)

## Customization Options

### Colors
Edit `styles.css` to change the color scheme:
- Primary color: `#2c3e50` (dark blue)
- Accent color: `#e74c3c` (red)
- Background: `#f8f9fa` (light gray)
- Text: `#333` and `#555`

### Fonts
Default font is "Segoe UI". To change, update in `styles.css`:
```css
font-family: 'Your Font Name', fallback-font;
```

### Sections
To add/remove sections, edit `index.html` and corresponding CSS in `styles.css`

## How to View the Website

### Option 1: Local Browser
1. Open `index.html` directly in your web browser

### Option 2: Local Server (Recommended)
1. Use VS Code Live Server extension
2. Or use Python: `python -m http.server 8000`
3. Visit `http://localhost:8000`
